Head vs Body — Simple Demo

What lives in <head>?

  • Title — shown in the browser tab (this page's title is inside <head>).
  • Meta — page description, charset, viewport, etc.
  • Link — stylesheets and icons.
  • Script — setup scripts that run before or after load.

These items are metadata; they are not directly visible inside the page body.

What lives in <body>?

The <body> contains the visible content: headings, paragraphs, images, forms, etc.

Example visible content: this text, images, and interactive controls.

Minimal code summary

<!DOCTYPE html>
<html>
  <head>…metadata like <title>, <meta>, <link>…</head>
  <body>…visible content like <h1>, <p>, <img>…</body>
</html>
